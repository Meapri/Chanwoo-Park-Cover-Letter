EdgeTranslate-v3: 실제 웹 페이지 번역을 견디는 MV3 라우팅

EdgeTranslate-v3는 페이지 번역을 단순 API 호출로 보지 않고, 브라우저 확장 안에서 여러 번역 경로와 UI 상태를 안정적으로 묶는 프로젝트입니다. Google/Page Translate, AI Page Translate, Gemini Nano, Chrome Built-in, Local/OpenAI 호환 경로를 분리해 라우팅하고, MV3 서비스 워커 환경에서도 IndexedDB 캐시와 Preact 패널 상태가 끊기지 않도록 설계했습니다.

구현 포인트

  • 번역 provider를 기능별로 나눠 웹 페이지 번역, AI 번역, 로컬 호환 API를 같은 흐름에서 선택할 수 있게 구성했습니다.
  • IndexedDB 기반 영구 캐시, URL 해시 프리패치, TTL 정리를 붙여 MV3 제약 안에서도 반복 번역 비용을 줄였습니다.
  • Preact 패널은 streaming result, floating/fixed 모드, docking, drag velocity, TTS까지 실제 사용 장면에 맞춰 다듬었습니다.
