PriType-Swift: macOS 입력기 경계 조건까지 다루는 한글 IME

PriType-Swift는 InputMethodKit과 libhangul을 연결해 한글 조합, 한자 후보, 상태 관리, 업데이트 흐름을 구성한 macOS 입력기입니다. 특히 Chromium/Electron 기반 앱에서 흔히 나타나는 delegate 수명과 커서 좌표 문제를 예외 케이스로 분리해 입력 안정성을 높이는 방향으로 설계했습니다.

구현 포인트

  • IMKServer와 libhangul 조합 상태를 분리해 입력 모드와 조합 라이프사이클을 추적하기 쉽게 만들었습니다.
  • 한자 사전 preload, 후보창, Right Option/Right Command 트리거를 입력기 경험 안에 연결했습니다.
  • Chromium/Electron 예외, cursor rect cache, force commit 처리를 별도 경로로 관리해 호환성 문제를 좁혔습니다.
